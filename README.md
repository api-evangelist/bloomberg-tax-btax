# Bloomberg Tax (BTAX)

Bloomberg Tax (BTAX) is a comprehensive tax research, planning, and compliance platform providing tax professionals with authoritative primary sources, expert analysis, and practical tools covering federal, state, and international tax law.

## APIs

- **Bloomberg Tax Data API** - Tax rates, regulations, and compliance data
- **Bloomberg Tax Transfer Pricing** - Transfer pricing comparables and documentation tools
